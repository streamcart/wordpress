WordPress Plugin for Streamcart

== Description ==

This plugin allow to connect the website with Streamcart. Then you'll be able to easily display your videos and lives
to engage your visitors.

#### What is Streamcart?

Streamcart was born to democratize access to Video and Live Shopping, bringing a platform that connects brands and customers through an immersive sales experience with lives and videos within the company's online store, transforming the customer journey.


== Installation ==

#### Installing the plugin
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Streamcart** and click "*Install now*"
2. Alternatively, download the plugin and upload the contents of `streamcart.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
3. Activate the plugin
4. Set [your Public Key](https://app.streamcart.io) in the plugin settings that will be available in the WordPress settings menu.

== Usage ==

#### Getting Started
Follow the steps below to set up and start using the Streamcart plugin on your WordPress site.

Step 1: Create Your Streamcart Account

1. Visit the Streamcart Website
- Go to https://streamcart.io/.

2. Sign Up
- Click on the "Sign Up" button.
- Fill in your details to create a new account.

Step 2: Install the Streamcart Plugin on Your WordPress Site

1. Log in to Your WordPress Admin Dashboard
- Access your WordPress site by logging in as an administrator.

2. Navigate to Plugins
- Go to the "Plugins" section in the left sidebar.
- Click on "Add New."

3. Search for Streamcart Plugin
- In the search bar, type "Streamcart" and press Enter.

4. Install and Activate the Plugin
- Click "Install Now" next to the Streamcart plugin.
- Once installed, click "Activate" to enable the plugin on your site.

Step 3: Integrate Streamcart with Your WordPress Site

1. Access the Integrations Menu
- Log in to your Streamcart account at (https://app.streamcart.io).
- Navigate to the "Integrations" menu.

2. Copy Your User Token
- Locate your streamcart-token in the Integrations menu.
- Copy the token for later use.

3. Insert User Token in WordPress
- Go back to your WordPress admin dashboard.
- In the Streamcart plugin settings, paste the copied user token into the designated field.
- Save the settings.

4. Verify Your Website
- Return to Streamcart App Site (https://app.streamcart.io) and follow the on-screen instructions to verify your website, completing the integration process.

Step 4: Start Using Video and Live Commerce

1. Create Your First Live Session
- Log in to your Streamcart account.
- Navigate to the "Lives" section.
- Click "Create New Live" to set up your first live commerce experience.
- Follow the prompts to configure your live session details.

2. Insert Store URL for Onsite Checkout
- In the checkout live settings, select the "Onsite" option.
- Enter your store URL to enable seamless checkout during live sessions.

3. Test Your Platform
- Conduct a test live session to familiarize yourself with the platform and ensure everything is functioning as expected.
