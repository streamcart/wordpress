WordPress Plugin for Streamcart

== Description ==

This plugin allow to connect the website with Streamcart. Then you'll be able to easily display your videos and lives
to engage your visitors.

#### What is Streamcart?

Streamcart was born with the purpose of democratizing access to Video and Live Shopping, bringing a platform that connects brands and customers through an immersive sales experience with lives and videos within the company's online store, transforming the purchasing journey.

== Installation ==

#### Installing the plugin
1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for **Streamcart** and click "*Install now*"
1. Alternatively, download the plugin and upload the contents of `streamcart.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin
1. Set [your Public Key](https://app.streamcart.io) in the plugin settings.
